## IP csv query
```sql
select distinct (ip.network) network,
       ip.isp isp,
       c.name country,
       n.name mno
from ip_db ip
inner join networks n on n.mcc = ip.mcc and n.mnc = ip.mnc
inner join countries c on c.id = n.country_id;
```
